1. Computers

Size
From small laptops to large mainframe systems.
Purpose
Personal computers, servers, supercomputers, etc.
Architecture
Design elements like memory organization or I/O systems.

Types of Computers

Type Description
Mainframe Large, expensive computers for high-speed data processing in large organizations.
Minicomputer Midrange computers for small businesses and industries, slower than mainframes.
Microcomputer Compact, low-cost computers with a microprocessor as CPU.
Desktop Personal computer designed to stay in one place, fits on or under a desk.
Workstation Specialized computer for high-performance tasks like video editing, CAD, and scientific modeling.

Smartphones

3. Features of Digital Devices

Figure 3: Features like portability, performance, and connectivity

Portability

Portable Devices
Laptop: Student carries to school for notes and assignments.
Tablet: Businessperson presents slides in meetings.
Smartphone: Tourist uses maps while traveling.
USB Flash Drive: Teacher carries lesson files between home and school.
Non-Portable Devices
Desktop Computer: Used at a fixed desk in office or home.
Television: Set up in living room for watching shows.
Printer: Kept in one place for regular use.

Performance

Processor Speed: More GHz = more instructions per second.
RAM: More RAM = more programs at once.
Storage Speed: SSDs are faster than HDDs.
Software Efficiency: Well-written code runs faster.
Example: A 3 GHz processor can execute 30 billion instructions in 10 seconds.
Example: 8 GB RAM, each program needs 500 MB: 8192/500 = 16 programs can run at once.
Example: 4 GB file, SSD speed 550 MB/s: 4096/550 ≈ 7.44 seconds to load.

Storage

Permanent data storage (HDD, SSD, USB, SD card, cloud).
More storage = more files, videos, software.
Faster storage = quicker loading and better performance.

User Interface

Command Line Interface (CLI): Text commands (e.g., Terminal).
Menu-Driven Interface: Navigate menus (e.g., ATM, old phones).
Graphical User Interface (GUI): Windows, icons, menus, pointers (e.g., Windows, macOS).
Voice Interface: Speak commands (e.g., Siri, Alexa).
Gesture Interface: Touch or motion (e.g., smartphones, Kinect).

Media Support

Supports various file types: images (.jpg, .png), audio (.mp3), video (.mp4), documents (.pdf).
Reads/writes from SD cards, USB, DVD, external drives.
Needs codecs and drivers for compatibility.

Energy Consumption

Amount of power used by devices.
High energy use = more carbon emissions.
Reduce by using efficient hardware, power-saving modes, and green computing.

Expansion Capability

Add hardware (RAM, storage, peripherals) to PCs.
Use ports (USB, HDMI) for external devices.
Mobile devices may support microSD cards.

Security Features

Software Security: Passwords, PINs, biometrics, timeout lock, retry limits.
Physical Security: Locks, tamper-resistant screws, alarm systems.

4. Types of Peripheral Devices

Figure 4: Examples of peripheral devices (keyboard, mouse, printer, webcam)

i) Input Devices

Keyboard: Typing letters, numbers, commands
Advantages: Familiar, accurate, widely supported
Disadvantages: Slower than voice input, can cause fatigue
Mouse: Controlling pointer, selecting items
Advantages: Easy to use, precise
Disadvantages: Needs flat surface, not ideal for mobility
Touchscreen: Direct interaction on screen
Advantages: Intuitive, no external device needed
Disadvantages: Screen smudges, less accurate
Microphone: Capturing voice input
Advantages: Hands-free, fast for speech
Disadvantages: May misinterpret due to noise
Scanner: Converting documents/photos to digital
Advantages: Preserves content digitally
Disadvantages: Bulky, slow for large volumes
Webcam: Capturing video
Advantages: Enables visual communication
Disadvantages: Privacy concerns, lighting affects quality
Joystick/Gamepad: Controlling games/simulations
Advantages: Responsive, good for gaming
Disadvantages: Limited use outside entertainment
Biometric Scanner: Scanning fingerprints, face, iris
Advantages: Fast login, high security
Disadvantages: Can fail with dirty fingers/poor lighting
Card Reader: Reading data from cards
Advantages: Quick input, used for access/transactions
Disadvantages: Vulnerable to cloning/skimming
Sensor: Detecting environment (temp, motion)
Advantages: Automated input, useful in IoT
Disadvantages: Expensive, needs calibration
