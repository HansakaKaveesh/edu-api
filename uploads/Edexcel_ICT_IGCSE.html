What is a digital device?

A digital device processes data using binary (0s and 1s). Devices can be grouped by type (computer, phone, tablet, console) and by functionality (input, output, storage, communication).

Computers
  • Mainframe
  • Minicomputer
  • Microcomputer
  • Desktop
  • Workstation
Mobile & consumer devices
  • Smartphones & specialist phones
  • Tablets
  • Cameras, consoles, smart TVs
  • GPS / navigation devices
Peripherals
  • Input (keyboard, mouse, scanner…)
  • Output (monitor, printer, projector…)
  • Storage (HDD, SSD, USB, optical…)

1. Computers

Computers can be classified by size, purpose, or architecture.

By size
  • Microcomputer → Desktop/Workstation
  • Minicomputer (midrange)
  • Mainframe
By purpose
  • Personal computing
  • Business/servers
  • Scientific/engineering
  • Mission-critical transactions
By architecture
  • Memory organization
  • I/O systems
  • Multi-user support
  • Parallel processing hardware

Computer types at a glance

Type Typical use Power Size Examples
Mainframe Massive transaction/data processing Very high (BIPS) Room/cabinet sized Banks, airlines, governments

Quick check

Which type best fits: airline reservations, student homework, 3D CAD design?

Key idea

Classification depends on the workload, number of users, and reliability needed.

Mainframe computers

Use
  • Large-scale data processing for banks, governments, airlines
  • Supports thousands of concurrent users/transactions
Examples
  • Financial transactions
  • Weather forecasting (large data processing)
  • Airline reservation systems
Key details
  • OS: IBM z/OS, Linux on zSystems
  • Size: room-sized / large cabinet systems
  • Power: exceptionally high (BIPS)
  • Advantages: reliable, secure, mission-critical
  • Disadvantages: expensive, trained staff required
  • Price: ~$500,000 to $10M+

Minicomputers

Use
  • Small to medium businesses
  • Database management and ERP systems
  • Multi-user environments
Examples
  • Research labs
  • Manufacturing control
  • University labs
Key details
  • OS: UNIX, Linux, VMS
  • Size: cabinet sized
  • Power: moderate–high (MIPS)
  • Advantages: cheaper than mainframes
  • Disadvantages: limited scalability; less common today
  • Price: ~$5,000 to $200,000

Microcomputers

Use
  • Web browsing, office work, gaming
  • Typically designed for a single user
Technology
  • Microprocessor-based CPUs (e.g., Intel Core, AMD Ryzen)
  • OS: Windows, macOS, Linux
Pros / cons
  • Advantages: affordable, widely available
  • Disadvantages: limited for intensive tasks
  • Price: ~$300 to $2,000
  • Applications: home use, small offices

Desktop computers

Use & examples
  • Document editing, gaming, light development
  • Office workstations, gaming PCs
Technology
  • Intel Core / AMD Ryzen (often with discrete GPUs)
  • OS: Windows, macOS, Linux
Pros / cons
  • Advantages: customizable, upgradable, long lifespan
  • Disadvantages: not portable; needs peripherals
  • Price: ~$500 to $3,000+

Workstations

Use
  • Video editing, CAD design, scientific modeling
  • Professional studios and engineering environments
Technology
  • Multi-core CPUs (Intel Xeon, AMD Threadripper)
  • Professional GPUs
  • OS: Windows Pro, macOS, Linux
Pros / cons
  • Advantages: reliability (ECC RAM), multi-monitor support
  • Disadvantages: high cost, needs expertise
  • Price: ~$2,000 to $10,000+

2. Mobile devices

Main categories
  • Mobile phones (smartphones, specialist phones)
  • Tablets
  • Consumer electronics (cameras, consoles, smart TVs)
  • Navigation devices (GPS)
  • Multifunctional / converged devices
Convergence

Convergence means combining the functions of several devices into a single device.

Smartphone example:
  • Telephone (calls)
  • Camera (photos/videos)
  • Music player
  • GPS navigation
  • Computer (web, email, apps)
  • Gaming console
  • Calendar/organiser
Why it matters: less to carry, cheaper overall, easier to sync/manage data.

Mobile phones

Smartphones
  • Apps, camera, GPS, internet, productivity tools
  • Applications: social media, shopping, mobile banking, gaming, work tasks
  • Disadvantages: battery life, fragility, cost
Specialist phones
  • Designed for specific needs (rugged, simple interface)
  • Advantages: durable, long battery life
  • Disadvantages: fewer advanced features
  • Applications: construction workers, hikers, elderly users

Tablets, consumer electronics & navigation

Tablets
  • Portable touchscreen; between phone and laptop
  • Applications: eBooks, e-learning, streaming, note-taking
  • Limitations: less powerful than laptops; fragile
Consumer electronics
  • Cameras/camcorders: high quality, optical zoom, 4K
  • Games consoles: optimized graphics + multiplayer
  • Home entertainment: smart TVs, home theatre
  • Media players: MP3/iPod (less common now)
Navigation devices (GPS)
  • Real-time directions and tracking
  • Applications: logistics, fleet management, hiking
  • Limitation: replaced by smartphone GPS apps

3. Features: portability

Portability

How easy a device is to carry and use in different locations (size + weight).

  • Portable: laptops, tablets, smartphones, USB flash drives
  • Non-portable: desktop PCs, televisions, many printers

Performance

A high-performance device runs tasks quickly and efficiently. Performance depends on hardware and software.

Main factors
  • CPU speed: executes instructions
  • RAM capacity/speed: supports multitasking
  • Storage speed: SSD loads faster than HDD
  • Software efficiency: well-written programs use fewer resources
Virtual memory (concept)

If RAM is full, the system uses storage as “virtual memory.” It keeps programs running, but it is slower than RAM.

Performance practice: calculations

Q1: Processor speed

A processor executes 3 billion instructions per second (3 GHz). How many instructions in 10 seconds?

3,000,000,000 × 10 =
30,000,000,000 instructions (30 billion)
Q2: RAM capacity

A computer has 8 GB RAM. Each program needs 500 MB. How many programs can run without virtual memory?

8 GB = 8 × 1024 = 8192 MB
8192 ÷ 500 = 16.384
16 programs

Virtual memory & storage speed

Q3: Virtual memory usage

A system has 16 GB RAM. If programs require 20 GB, how much virtual memory is needed?

20 GB − 16 GB =
4 GB virtual memory
Q4: Storage transfer speed

An SSD transfers at 550 MB/s. How long to load a 4 GB file?

4 GB = 4096 MB
4096 ÷ 550 = 7.4472… s
≈ 7.45 seconds
Q5: Multitasking

A computer has 16 GB RAM. Each program needs 1 GB. If it runs 20 programs, will it require virtual memory?

RAM required = 20 GB
Extra = 20 − 16 = 4 GB
Yes — 4 GB virtual memory

Storage

Storage is where files, programs, and data are saved (non-volatile storage keeps data when power is off).

Primary storage
  • RAM: volatile; stores active programs/data
  • ROM: non-volatile; stores startup instructions (e.g., BIOS)
Secondary storage
  • Non-volatile: keeps data when off
  • Stores documents and installed apps
  • Provides virtual memory when RAM is full
  • Examples: HDD, SSD, USB, SD/microSD, optical discs, cloud
Media vs device

Media = the material that holds data.
Device = the hardware that reads/writes the media.

  • Magnetic media → HDD platters, tapes
  • Optical media → CD/DVD/Blu-ray
  • Solid-state media → flash chips (SSD/USB/SD)
HDD (how it works)
  • Spinning platters + moving read/write head
  • Tracks and sectors organize data
  • Deleting often marks space “free” until overwritten
SSD & optical discs
  • SSD: flash memory, no moving parts, fast
  • Optical: pits/lands read by a laser (CD/DVD)
  • Rewritable discs: CD-RW/DVD-RW can erase and rewrite

User interfaces

A user interface (UI) is how a person interacts with a device.

1) CLI

Type commands (text-based).

Examples: Windows Command Prompt, macOS Terminal, Linux Terminal
2) Menu-driven

Choose options from menus.

Examples: ATMs, printer menus, car dashboard settings
3) GUI

Windows/icons/menus/pointers (WIMP).

Examples: Microsoft Windows, macOS, Android/iOS apps
4) Voice

Speak commands.

Examples: Siri, Alexa, Google Assistant
5) Gesture

Swipes/pinches/hand movements.

Examples: smartphone gestures, tablets, motion remotes
Compare UIs
  • CLI: fast for experts, hard for beginners
  • GUI: easy to use, needs more memory/storage
  • Voice: hands-free, can misinterpret speech

Connectivity, media support, energy

Connectivity

Devices connect to networks and other hardware using wired/wireless methods.

  • Ports: USB, HDMI, audio, Ethernet
  • Wireless: Wi‑Fi, Bluetooth, cellular data
Link to Unit 2: connectivity enables networks and shared resources.
Media support
  • File formats: .jpg/.png, .mp3/.wav, .mp4/.avi, .pdf/.docx
  • Storage media: SD/microSD, DVD, USB, external drives
  • Codecs & drivers: needed for compatibility
Energy consumption & green computing
  • Energy use affects cost and environmental impact
  • Reduce usage: sleep/hibernate, auto shutdown, switch off
  • Efficient hardware: LED monitors, SSDs
  • Green computing: reduce power use and e-waste

Expansion capability & security features

Expansion capability
  • Internal: add RAM, upgrade GPU, add SSD/HDD
  • External: connect devices via USB/HDMI
  • Mobile: some support microSD expansion
Security features
Software security
  • Passwords / PINs
  • Timeout lock, retry limits
  • Biometrics (fingerprint / face)
Physical security
  • Security slots/cable locks
  • Tamper-resistant screws
  • Locked rooms / alarms

4. Peripheral devices

Peripheral devices extend a computer’s functionality. They are grouped into: input, output, and storage.

Input

Sends data to the computer.

Keyboard, mouse, scanner, microphone, webcam, sensors…
Output

Displays results from the computer.

Monitor, printer, plotter, projector, speakers, actuators…
Storage

Stores data and programs.

HDD, SSD, USB flash drive, optical discs, SD cards…

Input devices

Device Use Advantages Disadvantages
Keyboard Typing text/commands Accurate, familiar Can be slow; fatigue
Mouse Pointer control in GUI Precise, easy Needs surface; not ideal for mobility
Touchscreen Direct screen interaction Intuitive; no extra device Smudges; less precise
Microphone Voice input / recording Hands-free; fast Noise; misinterpretation
Scanner Paper → digital Preserves documents/photos Bulky; slower for many pages
Webcam Video capture Video calls, streaming Privacy; lighting affects quality
Biometric scanner Identity verification Fast login; strong security Can fail; privacy concerns
Card reader Read ID/payment cards Quick transactions/access Risk of cloning/skimming
Sensors Detect environment changes Automation; IoT Cost; calibration errors

Output devices

Monitor
  • Displays text/images/video
  • Types: LCD/LED/OLED
  • Resolution: HD, Full HD, 4K
  • Can cause eye strain over long periods
Printers & plotters
  • Inkjet: great photos, slower
  • Laser: fast text, good for volume
  • 3D printer: builds objects layer-by-layer
  • Plotter: precise large drawings (maps/blueprints)
Projectors, speakers, control devices
  • Projector: large audience viewing (classrooms)
  • Speakers: audio output
  • Actuators: motors, valves, heaters
