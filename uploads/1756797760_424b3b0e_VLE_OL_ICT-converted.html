    1. ‌Computers


      Computers can be classified by size, purpose, or architecture:

      • Size: Computers can range in size from small laptops to large mainframe systems.

      • Purpose: Computers can be classified by their intended use, such as personal computers, servers, or supercomputers.

      • Architecture: Computers can be grouped by design elements like memory organization or input/output systems


      Mainframe

      Large, expensive computers that can process data at high speeds and are used by large organizations to manage large amounts of data.

      Minicomputer

      Also known as midrange computers, these computers are used by small businesses and industries and are slower than mainframes.

      Microcomputer

      A compact, low cost computer with a microprocessor as its central processing unit.

      Desktop

      A personal computer designed to stay in one place and fits on or under a desk.

      Workstation

      A specialized computer used for high performance tasks like video editing, graphic design, and scientific modeling.


      1. ‌Mainframe Computers


        Use:

        Large scale data processing for organizations like banks, governments, and airlines.


        Examples:

        Financial transactions, weather forecasting.


        Technology Used:

        High performance CPUs and specialized hardware for massive parallel processing.


        Operating Systems (OS):

        IBM z/OS, Linux on zSystems.


        Size:

        Room sized or large cabinet sized systems.


        Processing Power:

        Exceptionally high. Capable of processing billions of instructions per second (BIPS) .

        Supports thousands of concurrent transactions and users.

        Advantages:

        Handles enormous workloads with reliability. Superior data security.

        Ideal for mission critical tasks.

        Disadvantages:

        Very expensive.

        Requires highly trained personnel.

        Price:

        $500,000 to over $10 million.

        Real World Applications:

        Banking systems, census data management, airline reservation systems.


      2. ‌Minicomputers


        Use:

        Used by small to medium sized businesses for applications like database management and ERP systems.


        Examples:

        Research labs, manufacturing control.


        Technology Used:

        Multi user system architecture, slower than mainframes.


        Operating Systems (OS):


        UNIX, Linux, VMS.


        Size:


        Cabinet sized, smaller than mainframes.


        Processing Power:

        Moderate to high. Processes millions of instructions per second (MIPS) . Can support multiple users simultaneously.


        Advantages:

        Affordable compared to mainframes.

        Multi user capability.

        Disadvantages:

        Limited scalability for larger workloads. Becoming less common with modern servers.

        Price:

        $5,000 to $200,000.

        Real World Applications:

        Business servers, small research facilities, university labs.


      3. ‌Microcomputers

        Use

        Personal tasks like web browsing, office work, and gaming. Examples: Home computers, entry level desktops.

        Technology Used

        Microprocessor based, such as Intel Core or AMD Ryzen CPUs.

        Operating Systems (OS):

        Windows, macOS, Linux.

        Size:

        Compact, small enough to fit on a desk or in a home.

        Processing Power:

        Low to moderate. Processes millions of instructions per second (MIPS) . Designed for single user applications.

        Advantages :

        Affordable and widely available. Easy to use and maintain.

        Disadvantages :

        Limited power for resource intensive tasks. Less robust for multitasking.

        Price :

        $300 to $2,000.


        Real World Applications :

        Home use, small offices, basic computing tasks.


      4. ‌Desktop Computers

        Use :

        Professional and personal tasks like document editing, gaming, and light software development.

        Examples : Office workstations, gaming PCs.

        Technology Used :

        Intel Core, AMD Ryzen, or equivalent processors; often with discrete GPUs.

        Operating Systems (OS) :

        Windows, macOS, Linux.

        Size :

        Fits on or under a desk.

        Processing Power :

        Moderate to high. Can handle billions of instructions per second (BIPS) with modern CPUs. Suitable for multitasking and medium intensity workloads.

        Advantages :

        Customizable and upgradable. Long lifespan.

        Disadvantages :

        Stationary; not portable. Requires external peripherals.

        Price :

        $500 to $3,000+.

        Real World Applications :

        Office tasks, personal use, light gaming.


      5. Workstations Use :

      High performance tasks like video editing, CAD design, and scientific modeling. Examples : Professional studios, engineering workstations.

      Technology Used :

      Multi core CPUs (Intel Xeon, AMD Threadripper) and professional GPUs.


      Operating Systems (OS) :

      Windows Professional, macOS, Linux distributions.


      Size :

      Bulkier than desktops for additional cooling and power requirements.


      Processing Power :

      Very high. Can process tens of billions of instructions per second (BIPS) . Optimized for resource intensive, specialized applications.

      Advantages :

      Reliable with error correcting memory (ECC). Multi monitor support for enhanced productivity.

      Disadvantages :

      High cost.

      Requires technical expertise to maximize efficiency.

      Price :

      $2,000 to $10,000+.

      Real World Applications :

      Animation studios, engineering design firms, research labs.

    2. ‌Mobile Devices

      Here’s a descriptive note covering the categories, advantages, disadvantages, and real world applications of mobile and other related digital devices:


      1. ‌Mobile Phones

        Smartphones : Highly versatile devices that integrate cameras, GPS, internet access, and access to various apps. These devices support communication, entertainment, productivity, and navigation, making them indispensable for modern life. However, they tend to have short battery lives, are fragile, and can be expensive.

        Applications : Social media, online shopping, mobile banking, gaming, and work related tasks.

        convergence means combining the functions of several devices into a single device.

        Example with a Smartphone:

        A smartphone is a good example of convergence because it combines features of:

        • Telephone – Make and receive calls.

        • Camera – Take photos and videos.

        • Music player – Store and play songs.

        • GPS device – Navigation and maps.

        • Computer – Browse the internet, send emails, run apps.

        • Gaming console – Play games.

        • Calendar/organiser – Manage events and reminders.

          Why this is important in ICT:

        • Saves carrying multiple devices.

        • Reduces cost and space.

        • Easier to manage and sync data.


          Specialist Phones : Designed for specific purposes, such as rugged phones for outdoor use or simple models for basic communication. They offer long battery life and are durable but lack advanced features.

          Applications : Used by construction workers, hikers, and elderly users requiring simple interfaces.


      2. ‌Tablet Devices

        Tablets are portable touchscreen devices that bridge the gap between smartphones and laptops. They are ideal for media consumption, gaming, and light productivity tasks. Tablets are user friendly but are less powerful than laptops and can be fragile.

        Applications : Reading eBooks, online learning, video streaming, note taking, and casual gaming.

      3. ‌Consumer Electronics

        Cameras and Camcorders : These devices specialize in capturing high quality photos and videos. They provide advanced features like optical zoom and 4K recording, making them ideal for professionals. However, they are bulky and expensive compared to smartphones.

        Applications : Used by photographers, filmmakers, and content creators for professional shoots and live streaming.

        Games Consoles : Devices like PlayStation, Xbox, and Nintendo Switch are optimized for gaming with high quality graphics and multiplayer capabilities. While they are limited to gaming and entertainment, they are staples for eSports and home gaming setups.

        Applications : Gaming competitions, family entertainment, and virtual reality gaming.

        Home Entertainment Systems : These include smart TVs, home theaters, and surround sound systems. They offer immersive audiovisual experiences but can be expensive and require proper setup.

        Applications : Watching movies, gaming, and streaming music or videos at home.

        Media Players : Portable devices like MP3 players and iPods specialize in playing music and videos. While they are now less common due to smartphones, they still offer offline playback and long battery life.

        Applications : Used during workouts, travel, or in dedicated audio systems.


      4. ‌Navigation Devices

        Navigation aids like GPS systems provide real time location tracking and directions. Devices such as Garmin or in car navigation systems offer accurate guidance, particularly in vehicles and for outdoor activities. However, they may face obsolescence due to smartphone GPS apps.

        Applications : Logistics, fleet management, hiking, and travel.


      5. ‌Multifunctional Devices

        These devices combine multiple features into one, such as smartphones integrating a camera, GPS, and apps. While they save space and cost, multifunctional devices may wear out faster and are less specialized in individual functions.

        Applications : Smartphones are used for various purposes, including communication, navigation, and entertainment.


      6. ‌Converged Devices

        Technological convergence has led to devices like smartphones and tablets offering overlapping functionalities. For example, both can run apps, access the internet, and take photos.

        Convergence enhances convenience but may lead to redundancy.

        Applications : Smartphones and tablets are used interchangeably for communication, productivity, and multimedia tasks.


      7. ‌Special Phones

      Special phones are built for specific needs, such as rugged outdoor phones or phones with simple designs for elderly users. They offer durability and long battery life but lack modern features like internet access.

      Applications : Emergency communication, outdoor activities, and minimalistic use.

  • ‌Portability

    Portability refers to how easy it is to carry and move a device. A device is considered portable if it is small in size and light in weight, making it convenient to use in different locations.

  • Devices like laptops, tablets, and smartphones are designed to be portable.

  • For larger devices such as televisions or desktop computers, portability is not a key feature because they are meant to stay in one place.

    The portability of a device is important when users need to work or access information on the go.

  • ‌Performance


    A high-performance device is one that can carry out its tasks or run software quickly and efficiently.


    ‌Factors That Affect Device Performance:

    • Processor (CPU) Speed:

      • The processor executes software instructions.

      • A faster processor can carry out more instructions per second, improving overall speed.

    • RAM (Random Access Memory):

      • Holds active software instructions for quick access.

      • More RAM allows more programs to run at the same time.

      • Faster RAM means instructions are delivered to the processor more quickly.

    • Storage Speed (e.g., SSD or HDD):

      • Instructions are loaded into RAM from storage.

      • Faster storage leads to faster loading of instructions and data.

      • When RAM is full, the system uses virtual memory on the storage drive.

      • Fast storage improves performance when using virtual memory.

    • Software Efficiency:

      • Well-written software uses system resources efficiently.

      • Efficient code leads to faster processing and smoother performance.

‌Question 1: Processor Speed

A computer's processor executes 3 billion instructions per second (3 GHz). How many instructions can it process in 10 seconds?

Formula:

Instructions per second=Processor speed×Time (in seconds)Instructions per second=Processor speed×Time (in seconds)

‌Question 2: RAM Capacity

A computer has 8 GB of RAM. If each program requires 500 MB of RAM to run, how many programs can the computer run simultaneously without using virtual memory?

Formula:

Number of programs=Total RAMRAM required per programNumber of programs=RAM required per programTotal RAM

Solution:

The total RAM is 8 GB, and each program requires 500 MB.

Convert 8 GB to MB:

8 GB=8×1024=8192 MB8GB=8×1024=8192MB

Now, calculate the number of programs:

8192 MB500 MB=16.384500MB8192MB=16.384

So, the computer can run 16 programs without using virtual memory (since it can’t run a fraction of a program).

‌Question 3: Virtual Memory Usage

A system has 16 GB of RAM and 500 GB of storage (HDD). If the system uses virtual memory, and 4 GB of RAM are occupied, how much virtual memory space will the system need to use?

Solution:

If 4 GB of RAM are used up, the system will need to use 4 GB of virtual memory (since virtual memory is used when the physical RAM is full).

starts running 20 programs, how much RAM will be left, and will it require virtual memory?

  • ‌Storage

    Storage refers to the part of the computer where files, programs, and data are saved permanently (even when the computer is turned off).


    More storage capacity means more space to save documents, videos, software, and other files.

    Storage devices include Hard Disk Drives (HDDs), Solid-State Drives (SSDs), USB drives, SD cards, and cloud storage.

    Storage speed affects performance:

    • Faster storage means quicker loading of programs and files.

    • It also improves system responsiveness, especially when using virtual memory.


  • ‌User interface

A user interface is how a user interacts with a digital device. There are several types of user interfaces used in real life.



  1. ‌Command Line Interface (CLI)

    • Description: Users type specific commands using text.

    • Pros: Fast for expert users, uses less memory.

    • Cons: Hard to use for beginners.

    • Real-World Examples:

      • Windows Command Prompt

      • macOS Terminal

      • Linux-based systems (e.g. Ubuntu Terminal)



  2. ‌Menu-Driven Interface

    • Description: Users navigate through menus to select options.

    • Pros: Simple and beginner-friendly.

    • Cons: Can be slow due to many steps.

    • Real-World Examples:

      • ATM machines (banking menus)

      • Old mobile phones (e.g. Nokia)

      • Printer settings menu screens

      • Car dashboard settings menu



  3. ‌Graphical User Interface (GUI)

    • Description: Uses Windows, Icons, Menus, and Pointers (WIMP).

    • Pros: Easy to use, visually clear.

    • Cons: Requires more memory and storage.

    • Real-World Examples:

      • Microsoft Windows

      • Apple macOS

      • Google Chrome browser

      • Android and iOS apps



  4. ‌Voice Interface

    • Description: Users give commands by speaking.

    • Pros: Hands-free, useful for accessibility.

    • Cons: May misinterpret words, needs internet in many cases.

    • Real-World Examples:

      • Apple Siri (iPhones, iPads)

      • Amazon Alexa (smart speakers)

      • Google Assistant (Android phones, smart displays)

      • Voice-controlled car systems (e.g. Tesla)



  5. ‌Gesture Interface

  • Description: Uses finger or hand movements to control the device.

  • Pros: Fast and intuitive on touchscreens.

  • Cons: May not work well with dirty or wet hands.

  • Real-World Examples:

    • Touchscreen smartphones (swiping to unlock)

    • Tablets (pinch to zoom)

    • Smart TVs with motion remotes (e.g. LG Magic Remote)

    • Gaming systems like Xbox Kinect (hand movements to play games)

  • ‌Media support

    Media support refers to the capability of devices and software to handle different types of media files (text, images, audio, video) and storage formats.

    ‌Types of Media Support:

  • File Format Support

    • Devices/software must support different file types:

      • Images: .jpg, .png

      • Audio: .mp3, .wav

      • Video: .mp4, .avi

      • Documents: .pdf, .docx

  • Storage Media Support

    Devices must be able to read/write data from various storage types:

    • SD Card – Used in cameras and mobile phones

    • MicroSD Card – Smaller version for smartphones and tablets

    • DVD – Optical disc used for video, audio, and data storage

    • USB Flash Drive – Portable storage device

    • External Hard Drive – High-capacity backup and file transfer

  • Codecs & Drivers

    • Codecs are used to compress/decompress media files

    • Proper drivers are required for hardware compatibility

  • ‌Energy consumption


    Energy consumption refers to the amount of electrical power used by computer systems and digital devices during operation.

    ‌Key Points:

    • Devices that Consume Energy:

      • Computers & laptops

      • Monitors & printers

      • Servers & data centers

      • Mobile devices (charging)

  • ‌Environmental Impact:

    • High energy use increases carbon emissions

    • Contributes to global warming and climate change

  • ‌Reducing Energy Consumption:

    • Energy-efficient hardware (e.g., LED monitors, SSDs)

    • Use of power-saving modes (e.g., sleep or hibernate)

    • Automatic shutdown settings

    • Cloud computing to reduce need for local servers

    • Switching off devices when not in use

  • ‌Green Computing:

    • Designing and using computers in an environmentally friendly way

    • Minimizing power use and e-waste

  • ‌Security features

  1. Software Security

    Definition:

    Software security helps protect data on a device by restricting access to authorised users.

    🔒 Common Methods:

    • Passwords / PINs: Required to unlock a device.

    • Timeout Lock: Device locks after a period of inactivity.

    • Retry Limit: After several incorrect attempts, the system:

      • Delays further attempts

      • May erase data for extra protection

    • Biometric Security: Uses fingerprints or facial recognition.

      • Example: Apple Touch ID®, Face ID

        ✅ Purpose:

    • Prevent unauthorised access

    • Protect against data theft

    • Useful if a device is lost or stolen



  2. Physical Security

    Definition:

    Physical security protects the actual device from being stolen or tampered with.

    🛡 Methods:

    • Security slots and locks:

      • Devices can be locked to desks or furniture using special cables.

      • Prevents easy removal or theft.

    • Tamper-resistant screws:

      • Used in devices to prevent access to internal parts.

        🖥 Examples:

    • Laptop cable locks

    • Locked server rooms in offices

    • Alarm systems on computer racks


      Summary:

    • Software security protects data and system access.

    • Physical security protects the device itself.

    • Both are important for personal, school, and business use of technology.

      • Writing documents

      • Coding or data entry

        Advantages:

      • Simple and widely used

      • Precise and accurate input

        Disadvantages:

      • Slower than voice input

      • Can cause hand fatigue with long use


    2. ‌Pointing devices Definition:

      A mouse is a pointing device used to control the cursor and select items on a screen.

      Uses:

      • Navigating graphical interfaces

      • Selecting and dragging items

      • Drawing in design software

        Advantages:

      • Easy to use and precise

      • Useful for GUI (Graphical User Interface)

        Disadvantages:

    3. Needs flat surface

    4. Not suitable for all devices (e.g., tablets)

    5. ‌Scanner

    6. ‌Barcode scanner

    7. ‌Web cam

    8. ‌Microphone

      Definition:

      A microphone captures audio input (voice or sound) and sends it to the computer.

      Uses:

      • Voice commands

      • Audio recording

      • Online calls or gaming

        Advantages:

      • Enables hands-free control

      • Fast for voice input

        Disadvantages:

        1. May pick up background noise

        2. Speech recognition may not be 100% accurate

      A touchscreen allows users to interact directly with what is displayed on the screen by touching it.

      • No need for external input devices

        Disadvantages:

        2. Screen smudges and reduced visibility

      Common Types:

      • Fingerprint scanner

      • Facial recognition

      • Iris or retina scanner

      • Voice recognition



        Uses:

      • Unlocking smartphones (e.g., Apple Touch ID, Face ID)

      • Securing access to laptops or buildings

      • Time and attendance systems in workplaces

      • Airport immigration systems



        Advantages:

      • High security – difficult to forge or steal

      • Fast and convenient – no need to remember passwords

      • Personal and unique to each user



        Disadvantages:

    11. ‌Card readers Definition:

      A card reader is an input device that reads data stored on a card, such as a magnetic stripe card or a smart card, to verify identity or access information.

      Common Types:

      • Magnetic stripe card reader

      • Chip (smart) card reader

      • Contactless card reader (RFID/NFC)

        Uses:

      • ATM machines for bank transactions

      • Access control systems in buildings or hotels

      • Public transport ticketing systems (e.g., travel cards)

      • Point-of-sale (POS) systems for payment

        Advantages:

      • Easy to use and widely available

      • Can store large amounts of data securely (smart cards)

      • Quick verification process

        Disadvantages:

        1. Cards can be lost, stolen, or damaged

        2. Magnetic strips can wear out over time

        3. May require special readers for certain card types


    12. ‌Sensors Definition:

      • Motion sensor

      • Temperature sensor

      • Proximity sensor

        Uses:

      • Automatic screen brightness adjustment in smartphones

      • Security systems to detect movement

      • Weather stations to measure temperature or humidity

      • Robotics for navigation and obstacle detection

        Advantages:

      • Can operate automatically without human input

      • Enables devices to adapt to environmental conditions

      • Increases efficiency and convenience in systems

        Disadvantages:

      • Sensors can malfunction or give inaccurate readings

      • Some may be sensitive to environmental interference

      • May require regular calibration and maintenance

      Common Types:

      • LED monitor – LCD technology lit by light-emitting diodes for better brightness and energy efficiency.

      • Touchscreen monitor – Combines display and input functions using capacitive or resistive touch sensors.

        Resolution:

        Resolution is measured in pixels (e.g., 1920 × 1080). Higher resolution means more pixels and therefore sharper, more detailed images.

      • Common resolutions: HD (1280×720), Full HD (1920×1080), 4K (3840×2160).

      • Higher resolution is important for design, photo editing, and watching high-quality video, but it may require more powerful graphics hardware.

        Technology Used:

      • LCD (Liquid Crystal Display): Uses liquid crystals to control light from a backlight.

      • LED (Light Emitting Diodes): Uses LEDs for backlighting; brighter, thinner, more energy-efficient.

      • OLED (Organic LEDs): Each pixel emits its own light; offers higher contrast and better colours but is more expensive.

        Uses:

      • Viewing documents, videos, and images

      • Gaming and entertainment

      • Designing graphics and editing media

        Advantages:

      • Wide range of sizes and resolutions available

    4. Large, high-resolution monitors can be expensive

      Output devices that send signals to control machinery, systems, or processes.

      1. Also called actuators, these are components that cause an action in the physical world.

  3. ‌Storage

  1. ‌Primary storage

    RAM ROM


    Primary Storage (Main Memory)

  2. RAM (Random Access Memory):

    1. Volatile memory – loses data when power is off.

    2. Stores data and applications currently in use.

    3. Allows fast read/write access for the CPU.

  3. ROM (Read Only Memory):

    1. Non-volatile memory – retains data when power is off.

    2. Stores permanent instructions, e.g., BIOS.

    3. Data cannot usually be modified by the user.

  4. ‌Secondary storage


  • Non-volatile memory – keeps data when the system is turned off.

  • Uses:

    • Store documents and files for future use.

    • Store applications, ready to load into RAM.

    • Provide virtual memory when RAM is full.

      Storage Devices

  • Devices that store data or software for a computer.

  • Internal: Inside the computer (HDD, SSD).

  • External: Connected via ports/adapters (USB drives, optical discs).

  1. Storage Media Types (What stores the data)

    • The physical material or medium where data is actually recorded.

    • Examples:

      • Magnetic media – Hard disk platters, magnetic tapes.

      • Optical media – CDs, DVDs, Blu-ray discs.

      • Solid-state media – Flash memory chips in SSDs, USB drives, memory cards.



  2. Storage Devices (What reads/writes the data)

    • The hardware that reads from or writes to the storage medium.

    • Examples:

      • Hard Disk Drive (HDD) – uses magnetic platters.

      • Optical Disc Drive – reads/writes CDs, DVDs, Blu-rays.

      • Solid State Drive (SSD) – uses flash memory chips.

      • USB Flash Drive – portable device with solid-state memory.



        Simple way to remember:

    • Media = the stuff that holds the data.

    • Device = the machine that accesses the data.
