UNIT 1: DIGITAL DEVICES

1. COMPUTERS

Computers can be classified by Size (laptop vs mainframe), Purpose (server vs supercomputer), or Architecture (memory organization).

1. Mainframe Computers

Large, expensive systems used by banks and governments.

Use: Large scale data processing (Banks, Governments, Airlines).

Processing Power: Exceptionally high. Processes Billions of Instructions Per Second (BIPS).

Size: Room-sized or large cabinet-sized.

Operating Systems: IBM z/OS, Linux on zSystems.

  • Advantages: Handles enormous workloads, superior security, ideal for mission-critical tasks.
  • Disadvantages: Very expensive ($500k - $10m+), requires highly trained personnel.

2. Minicomputers (Midrange)

Use: Small to medium businesses (Database management, ERP).

Processing Power: Moderate to high. Processes Millions of Instructions Per Second (MIPS).

Size: Cabinet-sized, smaller than mainframes.

  • Advantages: Affordable compared to mainframes, multi-user capability.
  • Disadvantages: Limited scalability, becoming less common.
  • Price: $5,000 - $200,000.

3. Microcomputers

Use: Personal tasks (Browsing, Office, Gaming).

Technology: Microprocessor-based (Intel Core, AMD Ryzen).

OS: Windows, macOS, Linux.

  • Advantages: Affordable, easy to use, widely available.
  • Disadvantages: Limited power for resource-intensive tasks.
  • Price: $300 - $2,000.

4. Desktop Computers

Use: Professional and personal tasks, fits on/under a desk.

Processing Power: Moderate to high (BIPS). Good for multitasking.

  • Advantages: Customizable, upgradable, long lifespan.
  • Disadvantages: Stationary (not portable), requires external peripherals.

5. Workstations

Use: High performance tasks (Video editing, CAD, Scientific modeling).

Specs: Multi-core CPUs (Xeon), ECC Memory, Professional GPUs.

  • Advantages: Reliable, multi-monitor support, high performance.
  • Disadvantages: High cost ($2k - $10k+).

2. MOBILE DEVICES

Technological Convergence

A single device performing the functions of many.

Definition: Combining the functions of several devices into a single device.

Example (Smartphone): Combines Telephone, Camera, Music Player, GPS, Computer, Gaming Console, and Calendar.

Importance: Saves carrying multiple devices, reduces cost, easier data management.

Device Categories

  • Smartphones: Versatile (Apps, GPS, Internet). Essential but fragile with short battery life.
  • Tablets: Bridge between phone and laptop. Good for media, less powerful than laptops.
  • Specialist Phones: Rugged or simple (elderly users). Long battery, durable, limited features.
  • Phablets: Large smartphones.

Consumer Electronics

  • Cameras/Camcorders: Higher quality than phones (Optical zoom, 4K). Bulky.
  • Game Consoles: Optimized for gaming (PlayStation, Xbox).
  • Home Entertainment: Smart TVs, Surround Sound.
  • Navigation (GPS): Dedicated guidance (Garmin), accurate but often replaced by phones.

3. FEATURES OF DIGITAL DEVICES

Portability

How easy it is to carry and move a device.

Device Scenario
Laptop Student carries to school for notes.
Tablet Businessperson presents slides at a meeting.
Smartphone Tourist navigates with maps.
Desktop PC (Non-portable) Fixed at a desk.

Performance

Key Factors: CPU Speed, RAM Capacity, Storage Speed (SSD vs HDD), Software Efficiency.

Question 4: Storage Transfer Speed

SSD Speed: 550 MB/s. File Size: 4 GB.

User Interface (UI)

  1. CLI (Command Line): Text commands. Fast for experts, low memory. (e.g., Linux Terminal).
  2. Menu-Driven: Select options. Simple but slow. (e.g., ATM).
  3. GUI (Graphical): WIMP (Windows, Icons, Menus, Pointers). Intuitive. (e.g., Windows).
  4. Voice: Spoken commands. Hands-free. (e.g., Siri).
  5. Gesture: Movements. Intuitive. (e.g., Touchscreens, VR).

Other Features

  • Connectivity: Linking to other devices/networks.
  • Media Support: Compatibility with files (.mp3, .jpg) and codecs.
  • Energy Consumption: Power usage. Green computing minimizes e-waste and power use.
  • Expansion: Adding hardware (RAM, GPU) or using ports (USB, HDMI).
  • Security:
    • Software: Passwords, Biometrics, Encryption.
    • Physical: Kensington locks, tamper-resistant screws.

Types: LCD, LED, OLED.
Resolution: Measured in pixels (HD, 4K). Higher resolution = sharper image.

Printers

  • Inkjet: Liquid ink, good for photos, slower.
  • Laser: Toner, fast, good for text/office.
  • 3D Printer: Builds objects layer by layer.

3D Printing: FOR

  • Encourages creativity & innovation
  • Medical uses (prosthetics)
  • Supports small-scale business

3D Printing: AGAINST

  • Safety risks (weapons)

Other Output

iii) Storage Devices

Primary Storage (Main Memory)

  • RAM (Volatile): Temporary, fast, loses data when power is off.
  • ROM (Non-volatile): Permanent instructions (BIOS).

Secondary Storage

Permanent storage for files and software. Media = the material (disk). Device = the machine (drive).

1. Hard Disk Drive (HDD) - Magnetic

How it works: Spinning magnetic platters with a read/write head. Magnetizes regions to represent 0s and 1s.

Deletion: Data isn't removed immediately; the sector is marked "available" until overwritten.

2. Optical Discs (CD/DVD/Blu-ray)

How it works: Laser reads "pits" (indentations) and "lands" (flat areas). Reflection = 1, Scatter = 0.

Types: ROM (Read only), R (Write once), RW (Rewritable - uses reheat-able dye).

3. Solid State Drive (SSD) - Flash

How it works: Uses flash memory chips (transistors) with electrical charges. No moving parts.

Pros: Much faster, silent, more durable than HDDs.
